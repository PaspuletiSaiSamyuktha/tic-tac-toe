# Tic Tac Toe Game

This is a simple Python implementation of the classic Tic Tac Toe game.

## Features
- Two-player mode.
- 3x3 grid.
- Checks for wins and draws.

## How to Run
- Copy the code from `tic_tac_toe.py`.
- Run the script in any Python 3.6+ environment.
